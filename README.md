# Machine Learning Internship Tasks

This repository tracks my internship task submissions.

## Repository Structure

- `task1-student-performance/`: Regression and clustering on student performance data
- `task2-customer-clustering/`: Customer segmentation with K-Means and DBSCAN
- `task3-forest-cover-classification/`: Forest cover type classification with Random Forest and XGBoost
- `task4-loan-approval-classification/`: Loan approval prediction with imbalanced-data handling
- `task5-movie-recommendation/`: Movie recommendation system with collaborative filtering and SVD
- `task6-music-genre-classification/`: Music genre classification with tabular audio features and transfer learning
- `task7-walmart-sales-forecasting/`: Time-series sales forecasting with lag features and time-aware validation
- `task8-traffic-sign-recognition/`: Traffic sign recognition with custom CNN and MobileNet transfer learning
